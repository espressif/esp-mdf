Mlink API
============

``Mlink`` (MESH LAN communication protocol)  is a solution for controlling ESP-MESH network devices through APP, including: device discovery, control, upgrade, etc.

.. ------------------------- Mlink App Reference ---------------------------

.. _Mlink-application-examples:

Application Examples
---------------------

For Mlink examples, please refer to the directory :example:`development_kit`, which includes:

   * Smart light: Mesh App can control the color of the lamp, distribution network, upgrade, etc.
